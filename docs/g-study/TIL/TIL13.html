[항해99 취업 리부트 코스 학습일지] - TIL Day 13 알고리즘 주차 - 알고리즘 2주차 시작 & 스택, 큐, 덱

[2024.06.05]​

👩🏻‍💻 Today I Learned​

알고리즘&자료구조 2주차 시작이다.
오늘 새로 배운 스택(stack), 큐(queue), 덱(deque) 을 활용해서 문제를 풀어봤다.
저번주에 비해 문제 해석은 쉽게 되었지만 아직까지 코드로 작성하는 부분에서 오래 걸렸다.

stack​

  • 후입선출 (LIFO, Last In First Out) 방식
  • stack 안에 데이터를 넣고, 마지막에 넣은 데이터를부터 빼는 구조

queue​

  • 선입선출 (FIFO, First In First Out) 방식
  • 처음 들어간 데이터를 제일 먼저 빼는 구조

deque​

  • stack + queue
  • 양방향에서 데이터를 조작하는 구조

위 이미지와 예제 코드들을 보면서 돌아가는 구조에 대해 이해하는데 도움이 많이 되었다.
더 이해하고 풀이에도 적용해보자.
